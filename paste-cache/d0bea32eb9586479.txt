WRDS does not appear to offer a single, standalone “global stock screener” product; instead, it provides datasets and analytics tools where you can *build* global screens by querying the underlying global equity databases your school subscribes to (most commonly Compustat Global). [wrds-www.wharton.upenn](https://wrds-www.wharton.upenn.edu/pages/grid-items/compustat-global-wrds-basics/)

## What can be screened globally on WRDS
- **Compustat Global** on WRDS includes (at least) Fundamentals, Index constituents/prices, and Securities data, and WRDS explicitly discusses filtering by country as part of using Compustat Global—this is essentially the foundation for a global screener-style workflow. [wrds-www.wharton.upenn](https://wrds-www.wharton.upenn.eduhe goal is a global screener
The practical path on WRDS is usually: use Compustat Global (and/or World Indices constituents) to define the global universe, then filter on variables/ratios via WRDS query tools or SQL/Python/Stata after extracting the data.
