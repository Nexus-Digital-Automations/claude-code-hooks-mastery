Downie’s “Import from File…” feature lets you batch-add a list of downloads (URLs plus optional custom titles) from a CSV or XML file, and it then queues them for processing/downloading. You use it from the menu: **File > Import from File…** and follow the prompts. [software.charliemonroe](https://software.charliemonroe.net/help/downie/import.html)

It only works for links that Downie directly supports as a URL input; it’s not meant for cases where Downie would have to *discover* the media by analyzing a surrounding webpage (i.e., “extraction techniques”). For example, a direct YouTube embed URL can work, but a random webpage that merely embeds a YouTube video (and would require scanning the page source to find it) may not import successfully. [software.charliemonroe](https://software.charliemonroe.net/help/downie/import.html)

For file formats, Downie supports: [software.charliemonroe](https://software.charliemonroe.net/help/downie/import.html)
- **CSV**: must have two columns named `url` and `title` (other columns can exist but are ignored). Example header: `url,title`. [software.charliemonroe](https://software.charliemonroe.net/help/downie/import.html)
- **XML**: root element must be `<downloads>`, containing one or more `<download>` entries, each with `<url>` and `<title>`. [software.charliemonroe](https://software.charliemonroe.net/help/downie/import.html)
